# Copilot-demo — Social Feed (static)

This is a small static demo that shows a modern social-media-like feed built with plain HTML, CSS and JavaScript.

How to run

- Open `index.html` in your browser (double-click or via the browser File->Open).
- The feed includes sample posts. Use the composer at the top to add a new post. Click the heart to like.

Files

- `index.html` — main HTML layout
- `styles.css` — styles for layout and components
- `app.js` — feed rendering and interactivity

No build step required.
